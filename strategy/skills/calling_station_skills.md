# Calling Station 策略指南

## 概述
很少弃牌或加注，只是跟注看牌。

## 翻前
### 入池范围 (宽)
- **任意口袋对子**
- **任意AX**
- **任意KQ, KJ, QJ, JT**
- **任意同花两张牌**
- **任意顺子型两张牌**
- **几乎任意两张牌**

### 3-bet范围
几乎不3-bet，只在AA/KK时考虑。

### 面对加注
- **宽泛跟注**: 看翻牌
- **弃牌**: 几乎从不

### 面对4-bet
- **跟注**: AA, KK
- **弃牌**: 极少数情况

## 翻牌
### 持续下注 (20%)
极少下注，除非拿着超强牌。

### 过牌 (主要策略)
几乎总是过牌。

### 跟注 (主要行动)
- 任何对子
- 任何听牌
- 任何成牌
- **跟注**是核心策略

## 转牌
### 继续跟注
- 任何对子
- 任何听牌
- 任何牌力

### 面对再次下注
- **继续跟注**
- 几乎不弃牌

## 河牌
### 跟注到底
- 任何对子
- 任何看起来合理的牌
- **几乎总是跟注**

### 弃牌
极少，只有拿到垃圾牌时才弃。

## 关键原则
- 不加注
- 很少弃牌
- 总是看摊牌
- 很少诈唬
- 被动地跟注

## GTO 参考
### 翻前入池范围（宽）
- 几乎任意两张牌都可能入池
- 任意对子、任意 AX、任意同花
- 3-bet 后仍跟注看翻牌

### 翻前加注尺寸（BB为核心）
- 无人limp开池: UTG 2.5-3BB, MP 2.2-2.5BB, HJ/CO/BTN 2-2.2BB, SB 3-3.5BB
- 有人limp时隔离加注: 基础2BB + 每limp者+1BB
  - 0人limp: 2BB
  - 1人limp: 3BB
  - 2人limp: 4BB
  - 3人limp: 5BB
  - 5人limp: 7BB
- 3-bet: 几乎不做3-bet
- 最小加注: 必须比前次加注多2BB以上

### 翻后策略（非 GTO）
- C-bet 跟注站: 20-25% C-bet（他们会跟）
- 面对 C-bet: 55-65% 跟注
- C-bet 尺寸: 33%-50% pot
- 几乎不 raise（被动）

### 转/河牌
- 跟注到底: 任何对子，任何听牌
- 下注尺寸: 40%-55% pot
- 弃牌率: 极低（5-10%）
- 被 value bet 仍跟注

### GTO 对抗建议
- 用强牌价值下注到底
- 几乎不诈唬（他们会跟）
- 避免在危险牌面 value bet 弱牌
- 他们是最容易被剥削的风格

## 弱点利用
- 如果你激进，他们会一直跟
- 河牌价值下注会被跟
- 不要试图偷他们的盲注
